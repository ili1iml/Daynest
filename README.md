# Daynest-Daily Task Manager
Daynest is a sleek and modern daily task management web application. It features a clean neumorphic design with full support for 
Dark/Light modes, helping users organize thier daily schedules efficiently.

### ✨ Features | المميزات
* Task Management: Add, complete, and delete tasks easily.
* Local Storage Integration: Tasks are saved in the browser, so they stay even after refreshing. (حفظ المهام تلقائياً في المتصفح)
* Prioritization: Assign priority levels (High, Medium, Low).
* Dark Mode: Smooth transition between Dark and Light themes.
* Calendar View: Visual calendar to keep track of the date.

## 🛠️ Built With | التقنيات المستخدمة
HTML5 (Structure)
CSS3 (Styling & Animations)
JavaScript (Logic & Interactivity)
